# Operadores Aritméticos

- Soma: a + b | a.plus(b) | a+=b
- Subtração: a - b | a.minus(b) | a-=b
- Multiplicação: a*b | a.times(b) | a*=b
- Divisão: a / b | a.div(b) | a/=b
- Resto: a % b | a.mod(b) | a%=b
